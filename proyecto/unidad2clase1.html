Estructura de un programa en JAVA (clases, métodos, atributos, librerías)

Un programa en Java se estructura de la siguiente manera:

1. **Clases**: En Java, todo programa comienza con la definición de una o varias clases. Las clases son plantillas para objetos y contienen atributos y métodos. Cada programa de Java debe tener al menos una clase con un método `main` que actúa como punto de entrada.

2. **Métodos**: Los métodos son bloques de código que realizan tareas específicas. Cada clase tiene uno o más métodos. El método `main` es fundamental y se ejecuta al inicio del programa. Otros métodos realizan acciones específicas y pueden ser llamados por otros métodos o desde otras clases.

3. **Atributos**: Los atributos son variables que almacenan datos dentro de una clase. Estos datos representan el estado de los objetos creados a partir de la clase. Los atributos definen las características y propiedades de los objetos.

4. **Librerías**: Java ofrece una amplia variedad de librerías estándar que contienen clases y métodos predefinidos para realizar tareas comunes. Estas librerías simplifican el desarrollo de aplicaciones al proporcionar funcionalidades reutilizables. Algunas de las librerías más comunes son `java.util` (para estructuras de datos y utilidades), `java.io` (para entrada y salida), y `java.swing` (para crear interfaces gráficas).

En resumen, un programa en Java se estructura en torno a clases que contienen atributos y métodos. Los métodos definen la funcionalidad del programa, mientras que los atributos almacenan los datos. Además, se pueden aprovechar las librerías estándar de Java para simplificar el desarrollo al acceder a clases y métodos predefinidos que realizan tareas comunes. La clase principal debe contener un método `main` que sirve como punto de entrada del programa.
